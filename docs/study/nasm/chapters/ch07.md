# 第7章 私人训练营养学

## 一、章节核心知识点系统梳理
### 1. 核心概念界定
- 私人教练在营养上的首要任务，是在`执业范围内`提供一般性营养教育，帮助客户识别误导信息、建立更合理饮食习惯，并让饮食服务训练目标。
- 当客户涉及糖尿病、心脏病、胃肠疾病、饮食紊乱、骨质疏松、高胆固醇等营养相关疾病，或问题复杂度超出私人教练能力时，应转介给营养专业人士，这类干预属于`医学营养治疗`范围。
- `饮食评估`是提出饮食建议前的前提。教材强调不要先入为主地假设客户“吃得不错”或“吃得很差”，而是要先收集当前摄入信息。
- 常见摄入数据收集方法有三类：`饮食回忆`、`饮食历史`、`饮食记录`。三天饮食记录通常最有效，但也最依赖执行力，且记录行为本身可能压低真实摄入。
- “`我的餐盘`”是教材重点工具，它把健康饮食具象化为谷物、蔬菜、水果、蛋白质、乳制品五大食物组，并配套限制钠、饱和脂肪和添加糖的实践建议。
- `能量需求`不是一个精确值，而是基于静息代谢、体力活动和食物热效应做出的估算；受年龄、去脂体重、体力活动、环境温度和日常非运动活动等因素影响很大。
- 本章一再强调：没有适用于所有人的“`绝对正确饮食`”或“最佳饮食”。合适的饮食应该既能提供所需营养与能量，又能匹配客户的偏好、生活方式、预算和训练目标。
- 对备考来说，最重要的不是背流行饮食名字，而是会判断：`是否超出执业范围、是否先做评估、是否满足目标、是否可能造成营养不足、是否需要转介`。

### 2. 机制与原理
- 在做任何营养建议前，私人教练应先确认所在地对营养建议的法律限制。教材举例说明，不同地区对许可、认证、注册的要求不同，因此`先明边界、再给建议`是首要原则。
- 一个完整营养评估通常包含`饮食数据、人体测量数据、生化数据、临床检查`，虽然私人教练往往不负责全部内容，但应理解整体框架，以便与营养师协作。
- 饮食摄入数据收集有各自特点：饮食回忆操作最简便但依赖记忆；饮食历史可了解习惯、病史与偏好；饮食记录最细，但会因记录行为改变原本饮食模式。
- `我的餐盘`的核心不是死背分量，而是建立结构化饮食意识：五类食物都要覆盖，同时减少钠、饱和脂肪和添加糖。教材还给出具体实践：一半谷物选全谷物、重视完整水果、蔬菜多样化、选择低脂乳制品、蛋白质来源多样且偏瘦。
- 能量需求的三大来源是`静息代谢率 RMR、体力活动、食物热效应`。其中 RMR 对多数成年人总能量消耗贡献最大，通常约占 60%-75%。
- RMR 会受多种因素影响：去脂体重增加、年龄较小、生长发育、体温异常等可提高 RMR；低热量摄入、去脂体重减少和甲状腺功能减退可降低 RMR。
- 评估体力活动时，不能只看正式训练课，还要把`职业活动、久坐时间、非运动性身体活动`都算进去，否则总能量需求容易估低。
- 增肌增重的基本逻辑是`热量盈余 + 足够蛋白质 + 渐进抗阻训练`。教材指出，每天高于日常需求约 350-500 kcal，可支持每周约增加 1 磅瘦组织；块头已经较大的训练者，更适合较小热量盈余和较慢增重。
- 蛋白质摄入要与目标场景绑定。增重或减脂时，教材都强调`至少 1.6 g/kg/天`蛋白质；若主要依赖植物蛋白，可能需要更多，因为与肌蛋白合成相关的关键氨基酸含量较少。
- 减脂的底层机制始终是`负能量平衡`。教材给出常见策略：每周总体重下降约 1%，并选择低能量密度食物来提升饱腹感、帮助控制热量摄入。
- 对减脂客户，教材提醒要警惕极端做法：过低热量摄入可能导致脱水、维生素矿物质不足、瘦体重显著减少和训练表现下降。
- 膳食补充剂在法规上被定义为`食品而非药物`。根据教材，补剂可以宣称“支持结构或功能”，但不能宣称诊断、预防、治疗或治愈疾病。
- 评估补剂时，关键不是只看名字，而是看`成分来源、总摄入叠加、客户目标、药检风险`。教材提醒，维生素矿物质可能来自补剂、早餐麦片、能量棒、运动饮料等多个来源，存在过量风险。

### 3. 实践应用场景
- 对任何客户开口谈饮食之前，先做基础评估：当前怎么吃、是否有过敏或不耐受、是否在执行素食或生酮、是否节食、是否常吃零食、是否刚换饮食方式。这些都会直接影响建议方向。
- 对执行力强、愿意精细追踪的客户，可优先使用三天饮食记录或饮食分析软件；对大多数普通客户，饮食回忆或饮食历史往往更现实、更容易长期执行。
- 对普通健康客户，教练可以用`我的餐盘`做低门槛教育：一半盘子偏水果蔬菜、谷物中至少一半选全谷物、乳制品偏低脂、蛋白质来源多样化，同时减少添加糖与高钠加工食品。
- 对增肌客户，教练应把建议具体化为：增加进餐频率、提高单餐总热量、选择高热量食物或代餐，并确保蛋白质达到至少 1.6 g/kg/天，而不是笼统地说“多吃点”。
- 对减脂客户，教练应优先抓住四个杠杆：`形成热量缺口、控制减重速度、提高蛋白质摄入、提高低能量密度食物占比`，并结合训练安排避免瘦体重流失。
- 当客户被各种流行饮食吸引时，教练不需要逐个熟悉每种饮食名称，而应引导客户检查这类方案是否排除整类食物、是否热量过低、是否承诺快速减重、是否忽视运动。
- 对高训练量有氧客户，饮食建议应重点覆盖`碳水化合物与液体支持恢复`；对忙碌人群，则可评估代餐饮料、能量棒等是否有助于提高依从性。
- 当客户带来补剂瓶时，教练可以协助一起看标签、记录成分、识别可疑兴奋剂或重复摄入来源，但一旦涉及疾病、用药冲突或复杂方案，就应及时转介。
- 对准备参加药检的客户，教材特别提醒要重视补剂中的违禁风险，优先选择有相应认证标识的产品。
- 行为改变上，教材给出的是`以客户为中心、关注解决方案、逐渐增加营养丰富食物并逐渐移除高加工高能量食物`，而不是用高压、单向命令式方式推进饮食改变。

### 4. 易错点/重点警示
- `执业范围`是本章第一高频考点。私人教练可以做一般营养教育，但不能实施医学营养治疗，也不能超越当地法律界限给出疾病饮食处方。
- `先评估、后建议`是第二高频原则。没有饮食评估就直接给建议，容易把错误问题当成真正问题。
- `三天饮食记录最有效`并不等于最适合所有客户。它执行成本高，而且记录行为本身会抑制正常饮食，导致摄入量看起来低于真实水平。
- `能量需求是估算值`而不是精算值。不要把某个公式算出的卡路里视为绝对正确，更不能忽略非运动性活动和环境影响。
- `增肌`不是单纯多吃；需要热量盈余、足够蛋白质和抗阻训练共同作用。只增加热量而不训练，未必得到理想的瘦组织增长。
- `减脂`不是靠某个饮食名字起作用，而是靠负能量平衡起作用。凡是能减脂的饮食，本质上都是通过某种方式帮助形成热量缺口。
- `减重过快`往往伴随风险：脱水、微量营养素不足、瘦体重下降和运动表现下滑，因此教材不支持极端节食式方案。
- `补剂天然`不等于`补剂安全`。维生素矿物质也可能因为多来源叠加而过量，尤其铁、钙、锌、镁、烟酸、维生素 A 等更要警惕。
- `补剂是食品不是药物`，所以标签能说“有助于保持骨骼健康”，却不能说“预防骨质疏松”。这类结构/功能宣称与疾病治疗宣称要分清。
- 对饮食跟踪也不能一刀切。教材提醒，长期、密集追踪对一部分客户可能增加心理压力或饮食失调风险，因此是否追踪、追踪多久，都要与客户特征和目标匹配。

## 四、答案与解析
1. **正确答案：B** 解析：营养相关疾病和复杂问题应转介营养专业人士。出处概念：执业范围与转介。
2. **正确答案：A** 解析：私人教练可在合法范围内进行一般营养教育。出处概念：私人教练的营养角色。
3. **正确答案：D** 解析：常见饮食评估方法为饮食回忆、饮食历史和饮食记录。出处概念：饮食摄入数据收集。
4. **正确答案：C** 解析：三天饮食记录通常被认为最有效，但要求也最高。出处概念：饮食记录。
5. **正确答案：D** 解析：“我的餐盘”强调五类食物组，不包括酒精。出处概念：我的餐盘。
6. **正确答案：B** 解析：能量可用性指能否同时满足运动和生理功能的能量需求。出处概念：能量可用性。

8. **正确答案：C** 解析：减脂时建议至少 1.6 g/kg 蛋白质，必要时可更高。出处概念：减脂期蛋白质。
9. **正确答案：D** 解析：长时间耐力训练者推荐约 7-10 g/kg/天碳水化合物。出处概念：碳水化合物建议。
10. **正确答案：B** 解析：DSHFA 将膳食补充剂定义为食品而非药物。出处概念：补剂监管。
11. **正确答案：B** 解析：记录行为本身可能改变进食行为，使记录摄入偏低。出处概念：饮食记录缺点。
12. **正确答案：B** 解析：当前版“我的餐盘”更强调限制钠、饱和脂肪和添加糖。出处概念：我的餐盘要点。
13. **正确答案：B** 解析：体重稳定但功能下调，正是低能量可用性的典型情形。出处概念：能量可用性。

17. **正确答案：B** 解析：低能量密度食物体积大、热量较低，有助于控制饥饿和总摄入。出处概念：低能量密度食物。
18. **正确答案：B** 解析：补剂可宣称结构/功能影响，但不能宣称治疗疾病。出处概念：补剂标签法规。

20. **正确答案：B** 解析：饮食建议应适配个人喜好、生活方式、训练目标和预算。出处概念：饮食推荐原则。
