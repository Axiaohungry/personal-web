# 第18章 青春期前少年儿童、老年与孕妇客户

## 一、章节核心知识点系统梳理

### 1. 核心概念界定
- 本章聚焦三类需特殊处方思路的人群：青春期前少年儿童、老年人和孕妇。共同点是都能从有氧和抗阻训练中获益，但训练变量与注意事项必须因人群特征而调整。
- 青春期前是指出现第二性征前的阶段，教材给出的大致范围为：女孩 6-11 岁、男孩 6-13 岁。考试中常把这一年龄界定与儿童训练原则放在一起考查。
- 对儿童而言，训练目标不只是“消耗精力”，更是通过符合生长发育的体力活动提高有氧耐力、力量、柔韧性，以及灵活性、平衡、协调、反应时、速度和爆发力等与运动技能有关的能力。
- 对老年人而言，训练目标除了改善心肺和肌肉适能，还包括维持功能独立性、减少跌倒风险、提升日常生活能力和生活质量，因此不能只盯着“练肌肉”或“练减重”。
- 对孕妇而言，前提是无产科或医疗并发症，并且在开始或调整运动前先咨询医疗健康人员。教材强调，孕期推荐的是经过调整的有氧与抗阻训练组合，而不是单一类型运动。
- 产后数周被教材称为“第四期”，这一时期不仅关注体能恢复，还要关注睡眠不足、疼痛、疲劳、尿失禁、情绪和产后抑郁等综合问题。

### 2. 机制与原理
- 儿童不是“缩小版成年人”。教材明确指出，儿童自然活动模式具有随意性、间歇性和爆发性，强度会自发上升或下降，并伴随短暂休息，因此不宜直接套用成年人长时间持续有氧或目标心率区间训练模式。
- 儿童的生理特点也支持这一点。运动中其呼吸频率更高、潮气量更低、每搏输出量更低、心率更高，最大心率预测公式在儿童身上准确性有限，所以不宜用成人式心率区间严格控制儿童训练。
- 儿童存在一定“代谢非特异性”，即无氧和有氧能力的分化不像成年人那样明显，因此更适合在发育期接触多种活动形式，而不是过早单项化。
- 儿童抗阻训练增力的主要来源是神经适应，而不是显著肌肥大。教材特别指出，由于青春期前儿童缺乏足够雄激素水平刺激明显肌肥大，所以力量提升主要来自神经适应、协调改善和运动技能提高。
- 老年人训练的生理基础在于：抗阻训练可改善肌力、爆发力、骨骼肌功能、骨密度、平衡和日常功能；有氧训练可改善心血管健康；平衡训练可降低跌倒风险。
- 老年人群常伴随柔韧性下降、平衡差、视听功能下降、热耐受能力下降等特点，因此其训练设计要更多考虑安全、渐进、环境控制和沟通方式，而不是只看负荷数字。
- 孕期运动处方要适应五类变化：心血管、呼吸、力学、代谢和体温调节。正因为这些变化，教材推荐用 RPE 和谈话测试而不是单纯心率来监控运动强度。
- 孕期心血管变化使“心率-耗氧关系”发生变化，因此中等强度更适合用 RPE 13-14 或能正常交流来理解；呼吸系统变化会提高通气量和静息耗氧量，使孕妇更容易在训练中感到呼吸负担。
- 力学方面，孕中晚期子宫增大与重心变化会影响平衡和动作控制；关节松弛要求动作更慢、更可控；盆底肌训练尤其重要，因为它与姿势支持、分娩准备和尿失禁风险有关。
- 考试易混点：孕中期开始前就应逐步退出仰卧位训练；儿童力量增加主要是神经适应；老年人并非不能练力量和爆发力，而是要更谨慎地安排方式和强度。

### 3. 实践应用场景
- 儿童的日常处方核心是“每天至少 60 分钟 MVPA”。形式应多样且有趣，可包括积极玩耍、户外游戏、步行或骑车上学、家务活动、循环训练、抗阻训练和团体运动，而不是局限于单调慢跑。
- 儿童课程设计更适合 8-12 站点的小型循环训练，内容可结合自重练习、绳类、药球、平衡练习和折返跑。随着体适能提高，再减少站间休息、增加任务挑战。
- 儿童抗阻训练开启方式应保守：中低负重、1-2 组、每组 8-12 次是安全有效的入门形式；随着技能提高再增加组数、负荷和复杂度。每周宜 2-3 天、非连续进行，并包含上肢、下肢和核心练习。
- 儿童执教关键不是“最大重量”，而是密切监督、安全器械、动态热身、动作示范、趣味性和持续指导。若多人同时训练，应按个体能力设定工作量，而不是给所有儿童统一负荷。
- 老年人的有氧训练可选步行、骑车、游泳等低冲击形式；抗阻训练通常从较轻到中等负荷起步，再逐渐增加；若平衡能力较差，应加入专门平衡训练，并由稳定姿势逐渐过渡到更具挑战性的任务。
- 老年人抗阻训练的执行重点包括：全 ROM 或无痛 ROM、持续呼吸避免屏气、较清晰的口头提示、较大字号训练卡、必要时提供手法辅助，并经常确认客户是否理解动作与流程。
- 孕妇运动方案应包括热身、放松、有氧训练、适度抗阻训练和盆底肌训练。适合贯穿孕期的低风险活动包括步行、游泳和某些团操；抗阻训练可更多使用坐姿、器械或阻力带以提高稳定性。
- 孕中晚期应逐步去掉仰卧位练习，如卷腹、卧推、仰卧稳定球练习和仰卧位拉伸，可改为四点支撑、侧卧位或坐姿练习。
- 孕晚期因重心变化更明显，可更多利用固定器械获得支撑；同时避免跌倒风险高、可能轻微伤及腹部、或需要高平衡控制的运动，如足球、篮球、骑马、潜水、直排轮、户外自行车和增强式训练。
- 产后在无并发症情况下可以逐步恢复体力活动。教材给出的大致时间是自然分娩后约 4-6 周、剖宫产后约 8-10 周，并在获医疗许可后渐进恢复；产后也应继续加入盆底肌训练，并重视中等强度活动对产后抑郁症状的缓解作用。

### 4. 易错点/重点警示
- 不能把成人训练理论强加给儿童。尤其是“目标心率区间持续 30 分钟以上”的模式，不应被默认为儿童最合适的方式；教材更支持游戏化、循环化和间歇化安排。
- 儿童抗阻训练并非禁忌，真正的风险来自缺乏监督、器械使用不当、训练量与难度超出承受能力，以及过早追求大重量或竞争比较。
- 儿童训练中应宁可低估能力、从简单开始，也不要高估能力。教材明确提示，超出儿童承受能力的训练量、强度和复杂度会削弱兴趣并提高受伤风险。
- 老年人训练中需特别避免屏气和瓦式呼吸，因为这会增加胸内压和血压负担；若存在平衡差、视听障碍、热耐受下降、柔韧性差等问题，教练要优先调整环境、器械和提示方式。
- 孕妇不应以“练到力竭”为目标，训练应维持在舒适、可交流的水平；出现过热、严重头痛、头晕、定向障碍等情况时，应立即降强度并必要时转诊。
- 孕期训练禁忌与预警是高频考试点。出现阴道出血、腹痛、规律宫缩疼痛、羊水渗漏、运动前呼吸困难、头晕、头痛、胸痛、肌无力影响平衡、小腿疼痛或肿胀，都需要停止运动并转诊。
- 对孕妇而言，仰卧位、过度拉伸、炎热潮湿环境、热瑜伽/热普拉提、可能伤及腹部或易跌倒的运动都需要避免或谨慎处理。
- 备考易混点：孕妇每周至少 150 分钟中等强度运动；儿童每天至少 60 分钟 MVPA；儿童增力主要靠神经适应；老年人不仅能练有氧，也能练抗阻、爆发力和平衡。
